# Rum Desktop App

[中文说明](README_cn.md)

## Requirements
- node 14-lts or greater
- go latest version
- yarn `npm install yarn -g`

## Start
Clone this repo and cd into it
```sh
yarn install
yarn build:quorum
yarn start
```
